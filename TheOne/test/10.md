TC-010: Validator Flags PLG Motion with Enterprise ICP
Purpose

Ensure V-SALES-01 triggers high severity warning.

Preconditions

ICP selected indicates enterprise, procurement buyer

Test Data

ICP company_size: 500+

budget_owner: procurement

sales_motion: plg

Steps

Set ICP to enterprise

Set sales motion to PLG

Run validator

Expected Results

Contradiction created: rule_id = V-SALES-01, severity = high

UI shows warning with recommended fix: switch sales motion or add enterprise sales support plan
