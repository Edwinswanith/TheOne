TC-007: Decision Ownership Enforcement
Purpose

Ensure only orchestrator writes decisions.*.selected_option_id.

Preconditions

At least one agent attempts to write decision selection directly (simulated or via fault injection)

Steps

Inject agent output patch that tries: replace /decisions/icp/selected_option_id

Run merge

Expected Results

Patch is rejected or ignored

Orchestrator records a merge warning in telemetry

Final decision remains unset until user selects or orchestrator sets it in allowed flow

Data Integrity Checks

telemetry.errors includes a structured error with component merge or similar

No state corruption occurs
